# Java Básico ☕

Este repositorio contiene ejemplos y conceptos fundamentales de **Java**, ideales para quienes están comenzando en el mundo de la programación.  

## Contenido
- Introducción a Java y su sintaxis
- Variables y tipos de datos
- Estructuras de control (if, switch, bucles)
- Funciones y métodos
- Clases y objetos
- Manejo de arrays y cadenas
- Ejemplos prácticos paso a paso

## Objetivo
El propósito de este proyecto es servir como guía de referencia rápida para estudiantes y desarrolladores que quieran **aprender Java desde cero** y practicar con ejemplos sencillos.

## Ejemplo rápido
```java
public class HolaMundo {
    public static void main(String[] args) {
        System.out.println("¡Hola, mundo!");
    }
}
